Meta-Learning with Learning Rate Control for Few-Shot Drug-Target Binding Affinity Prediction

📋 Overview
MetaDTA is a meta-learning deep learning model  for predicting kinase-drug binding affinity, based on an improved graph neural network architecture, optimized for high-performance computing environments.

🖥️ System Requirements
Recommended Configuration (Tested)
GPU: NVIDIA A800 80GB (currently using 1 card)

CPU: Intel Xeon Gold 6348 (14 cores)

Memory: 100GB RAM

Storage: 50GB available space (expandable)

CUDA: 13.0

Driver: 580.82.07 or higher

OS: Linux (Ubuntu 20.04/CentOS 7+)
⚙️ Quick Start
1. Environment Setup
# Clone repository
git clone  https://github.com/ljatynu/MetaDTA.git
cd MetaDTA

# Create and activate conda environment
conda create -n metadta python=3.9
conda activate metadta

# Install PyTorch (compatible with CUDA 13.0)
pip install torch==2.0.1+cu117 torchvision torchaudio --index-url https://download.pytorch.org/whl/cu117

# Install other dependencies
pip install -r requirements.txt

# Set Python path
export PYTHONPATH=$PWD:$PYTHONPATH
2. Data Download and Preparation
# Download example dataset (~120MB)
wget https://www.dropbox.com/s/owc45bzbfn05ix4/data.tar.gz

# Extract data files
tar -xzf data.tar.gz

# Verify data integrity
python scripts/check_data.py --dataset davis --dataset kiba

📊 Dataset Structure
Davis Dataset (data/DAVIS/)
davis_data.tsv: Kinase-drug binding affinity data

davis_protein2pdb.yaml: Mapping from kinase names to PDB structures

davis_cluster_id50_cluster.tsv: Protein clustering results

Data format example:

drug	protein	Kd	y
5291	AAK1	10000	5

KIBA Dataset (data/KIBA/)
kiba_data.tsv: Kinase-inhibitor binding affinity data

Uses ChEMBL IDs and UniProt IDs as identifiers

Structure Data (data/structure/)
pockets_structure.json: Representative kinase pocket structures

davis_moil3d_sdf/: 3D molecular structures for Davis dataset (SDF format)

kiba_moil3d_sdf/: 3D molecular structures for KIBA dataset (SDF format)

Pretrained Embeddings (data/esm1b/)
Protein sequence embeddings generated by ESM-1b model (.pt format)

🚀 Model Training
Run the MetaDTA Model
Execute the following script in the scripts/ directory. This script trains and evaluates the MetaDTA model using the Davis dataset of kinase-drug binding affinity:

bash
CUDA_VISIBLE_DEVICES=0 python run_example.py --task davis --n_epochs 100 --output_dir ../output/davis --save_prediction
Training Details
Dataset: Davis kinase-drug binding affinity dataset

Data Split:

70% as training data

10% as validation data

20% as test data (random split)
Output
After training and testing are finished:

Evaluation metrics will be printed in the terminal

Predicted binding affinity values will be saved as a TSV file in the output directory
